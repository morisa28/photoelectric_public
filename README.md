# 光电效应虚拟实验平台公开资料

本仓库用于公开展示光电效应虚拟实验平台项目中的部分过程资料和工作记录，便于了解项目调研、物理模型梳理、阶段记录和文档整理情况。

需要说明的是，本仓库仅作为公开资料仓库，不是完整工程仓库，不包含具体业务代码、核心工程文件、私有配置、部署脚本、接口密钥或其他不适合公开的内容。项目实际实现、工程结构和核心功能文件不在本仓库中维护。

## 内容范围

- `document/`：项目文档、物理框架说明、阶段记录、实施指导和写作材料。
- `knowledge/`：围绕光电效应实验整理的论文索引、主题卡片和知识映射。
- `papers/`：与项目论证相关的公开论文资料及元数据记录。

## 仓库定位

本仓库主要用于：

- 展示项目推进过程中的文档工作量。
- 保存公开范围内的物理模型、资料索引和阶段说明。
- 作为项目展示、答辩或过程复盘时的辅助材料。

本仓库不用于：

- 运行或部署完整项目。
- 复现核心工程实现。
- 发布生产代码或内部工程资产。

如需了解项目背景，可优先查看 `document/总目录/` 和 `knowledge/README.md`。
