# 当前模型标准场景数值核对表-new(26)

## 1. 核对目的

本表记录当前模型的标准场景数值。当前版本已经采用 A 路径：截止电压模式下加入有效阴极尾流，用于在不改阳极寄生电流公式、不改电流屏显示口径的前提下，让测量电流零点回到理论截止电压附近。

核对目标：

- 确认基础截止门是否仍在理论截止条件下关闭。
- 确认有效阴极尾流是否把测量电流零点拉回理论截止点附近。
- 确认阳极寄生电流对截止模式读数的影响量级。
- 确认 `-1.998 V` 强反压读数仍保留当前阳极寄生量级。

## 2. 标准场景

统一使用以下条件：

```text
阴极材料：Cs
滤光片：365 / 405 / 436 / 546 / 577 nm
光阑：4 mm
光电管距离：40 cm
测量模式：截止电压
阳极寄生：当前默认强度
暗电流：不进入 3D 仪器读数
本底电流：不进入 3D 仪器读数
仪器基准：按调零后接近 0 理解
```

电流拆分口径：

```text
有效阴极电流 >= 0
物理测量电流 = 有效阴极电流 + 阳极寄生电流
仪器显示电流 = 物理测量电流 + 仪器基准
```

下表中的“1e-13 A 档读数”按仪器基准为 0 时估算，用于观察量级。

说明：截止模式 A 路径下，“有效阴极电流”包含局部尾流；基础截止门在 `U=-Vs` 处仍为 0，仍用于截止质量判定。

```text
U = -Vs
```

- 这不是修改阳极寄生电流，而是修改截止模式下参与测量电流合成的有效阴极电流。

- 当前默认较明显的阳极寄生仍会让强反压下仪器读数偏负。

## 6. 对后续实施的含义

根据用户最新确认方向，本阶段尝试 A 路径：不回退整体物理框架，不新增特殊显示标准，不修改阳极电流公式，而是在截止模式阴极有效电流中加入局部尾流。

后续更合适的处理方向是：

1. 保留当前非包围式阴极收集率模型。
2. 保留当前 3D 电流屏显示仪器读数的口径。
3. 保留当前较明显的阳极寄生演示。
4. 截止记录继续保存用户手动外加反向电压。
5. 截止质量判定使用基础截止门，而不是有效尾流后的收集率。
6. 拟合只使用质量合格的手动截止记录。
7. 若正常操作仍明显偏离理论值，应继续检查电压精度、记录流程、调零基准和显示引导。

## 7. 当前阶段结论

当前模型标准场景核对显示：

- 基础截止门逻辑仍符合“截止后非负归零”的基本要求。
- 当前阳极寄生电流在 `-1.998 V` 下维持 `10e-13 A` 量级。
- A 路径使表观零电流点回到理论截止电压附近。
- 截止质量判定和拟合仍应依赖基础截止门与手动电压偏差，不应把有效尾流误认为理论截止门未关闭。
