## 1. 项目中的物理对象

当前项目模拟的是典型光电效应虚拟实验。物理对象可以按实验装置理解为：

1. 汞灯或光源提供入射光。
2. 滤光片选择入射光频率。
3. 光阑控制入射光束通量。
4. 光源到光电管的距离影响照射强度。
5. 阴极材料决定逸出功和阈频。
6. 光电管两极之间加外电压。
7. 阴极发射光电子，部分电子被阳极收集形成光电流。
8. 电流计显示仪器测得的电流。
9. 3D 区和原理动画把上述过程可视化。

项目中有两类演示状态：

- 实验仪器状态：用于 2D 实验台、3D 仪器、电流屏、记录和数据分析。
- 原理动画状态：用于独立讲解物理现象，可以单独调节频率、光强、电压、污染、暗电流和本底电流。

二者的控制入口可以不同，但底层物理思路保持一致。

## 2. 基本输入量

当前物理框架主要使用以下输入：

| 输入量 | 物理含义 | 对结果的主要影响 |
|---|---|---|
| 入射光频率 | 单个光子的能量来源 | 决定是否能逸出、最大初动能和截止电压 |
| 入射光强或光通量 | 单位时间到达阴极的光子数量 | 决定饱和电流大小 |
| 阴极材料逸出功 | 电子逸出金属表面所需最小能量 | 决定阈频和截止电压 |
| 外加电压 | 两极间电场强弱和方向 | 决定电子收集率 |
| 光阑大小 | 光束截面积 | 影响相对光通量 |
| 光源距离 | 光照几何衰减 | 按平方反比影响相对光通量 |
| 阳极污染程度 | 阳极表面产生寄生电子的可能性 | 在反向电压下贡献负向寄生电流 |
| 阳极漏光比例 | 杂散光照到阳极的程度 | 与阳极污染共同决定寄生电流 |
| 仪器零漂和调零 | 电流计本身的基准偏移 | 影响电流屏读数，不改变真实光电流 |

## 3. 光电效应基本理论

### 3.1 光子能量

项目采用爱因斯坦光电效应方程。单个光子的能量为：

```text
E = hν
```

其中：

- `E`：光子能量。
- `h`：普朗克常量。
- `ν`：入射光频率。

项目内部常用电子伏特作为能量单位，这样更便于和材料逸出功、截止电压对应。

### 3.2 阈频

材料逸出功记为：

```text
φ
```

当光子能量刚好等于逸出功时：

```text
hν0 = φ
```

因此阈频为：

```text
ν0 = φ / h
```

如果：

```text
ν <= ν0
```

则单个光子的能量不足以使电子逸出。此时无论光强多大，都不会产生正常阴极光电流。

### 3.3 最大初动能

当频率高于阈频时，电子逸出后的最大初动能为：

```text
Kmax = hν - φ
```

若频率低于阈频，则取：

```text
Kmax = 0
```

所以完整写法为：

```text
Kmax = max(0, hν - φ)
```

### 3.4 截止电压

截止电压的物理意义是：用反向电场刚好阻止最大初动能电子到达阳极。

能量关系为：

```text
eVs = Kmax
```

其中：

- `e`：元电荷。
- `Vs`：截止电压的数值。

当 `Kmax` 使用电子伏特表示时，截止电压以伏特表示时数值相等：

```text
Vs = Kmax
```

重要结论：

- 截止电压只由入射光频率和材料逸出功决定。
- 截止电压不由光强决定。
- 提高光强会增加光电子数量，但不会增加单个光电子的最大初动能。

## 4. 材料与频率

当前项目可选阴极材料包括 Ag-O-K、Cs、K、Na、Zn、Cu 等。不同材料的核心差异是逸出功不同。

逸出功越小：

- 阈频越低。
- 同一频率下更容易发生光电效应。
- 同一频率下最大初动能更大。
- 理论截止电压更大。

当前项目默认阴极材料为 Ag-O-K。这个默认值更接近常见教学光电效应仪器的复合光电阴极实测区间；Cs 仍保留为可选材料。

滤光片提供离散频率，例如紫光、蓝紫光、绿光和黄光等档位。项目中频率越高，光子能量越高，因而更容易超过材料逸出功。

## 5. 光通量与饱和电流

### 5.1 相对光通量

实验仪器状态中，光通量按光阑面积和距离平方反比计算：

```text
G = T · (D / D0)^2 · (d0 / d)^2
```

其中：

- `G`：相对光通量。
- `T`：滤光片透过率，当前按 1 处理。
- `D`：当前光阑直径。
- `D0`：参考光阑直径。
- `d`：当前光源到光电管距离。
- `d0`：参考距离。

物理含义：

- 光阑直径变为 2 倍，光阑面积变为 4 倍，通量约变为 4 倍。
- 距离变为 2 倍，照射强度按平方反比约变为 1/4。

在参考距离不变时，2 mm、4 mm、8 mm 光阑的通量比例约为：

```text
1 : 4 : 16
```

### 5.2 饱和电流

饱和电流表示在足够强的正向电压下，几乎所有可收集电子都被收集时的电流上限。

基础近似为：

```text
Is ∝ G
```

也就是：

```text
Is = Iref · η · G
```

其中：

- `Is`：饱和电流。
- `Iref`：参考条件下的电流尺度。
- `η`：等效量子效率。
- `G`：相对光通量。

为了模拟实际仪器量程和器件响应，当前项目还允许饱和电流随通量增大时逐渐接近一个仪器上限，而不是无限线性增大：

```text
Is = Ilimit · [1 - exp(-Ilinear / Ilimit)]
```

其中：

- `Ilinear` 是按光通量线性得到的理想电流。
- `Ilimit` 是仪器或器件近似上限。

## 6. 非包围式光电管的收集率

### 6.1 为什么需要收集率

真实光电管中，电子从阴极逸出后不一定全部到达阳极。是否被收集取决于：

- 电子初动能。
- 出射方向。
- 两极几何结构。
- 外加电压方向和大小。

当前项目采用“非包围式光电管”近似。它的特点是：

- 0 V 附近并不立即饱和。
- 即使没有正向加速电压，仍有少量电子可凭初动能到达阳极。
- 正向电压逐渐增大时，收集率逐渐接近 1。
- 反向电压逐渐增大时，收集率逐渐接近 0。

### 6.2 收集率的分段模型

设外加电压为 `U`，理论截止电压为 `Vs`，收集率为 `C(U)`。

当没有有效光电效应时：

```text
C(U) = 0
```

当反向电压达到或超过截止条件时：

```text
U <= -Vs
C(U) = 0
```

当处于反向电压但尚未完全截止时：

```text
C(U) = C0 · [(U + Vs) / Vs]^a
```

其中：

- `C0` 是 0 V 附近的基础收集率。
- `a` 控制反向区下降曲线的形状。

当电压为 0 V 或正向电压时：

```text
C(U) = C0 + (1 - C0) · [1 - exp(-(U / Ur)^b)]
```

其中：

- `Ur` 是伏安曲线正向上升的特征电压。
- `b` 控制正向区曲线形状。

当前框架中，0 V 附近基础收集率取较小值，使曲线不会在 0 V 附近直接饱和。

### 6.3 截止模式下的有效阴极尾流

截止电压测量模式下，当前项目还引入了一段只在截止模式使用的有效阴极尾流。它不改变基础截止电压公式：

```text
Vs = Kmax
```

也不改变基础收集率在理论截止条件下应关闭的判断：

```text
U <= -Vs 时，基础收集率为 0
```

但在实验仪器读数中，如果阳极寄生电流较明显，单纯让阴极电流在 `-Vs` 处等于 0，会导致电流表在理论截止点仍显示负值，用户若按“电流表归零”操作，就会停在偏低电压。

为避免这种操作性偏差，截止模式下使用有效收集率：

```text
C有效 = C基础 + C尾流
```

其中尾流强度按理论截止点处的阳极寄生电流反推，使：

```text
U = -Vs 附近
I有效阴极 + I阳极寄生 ≈ 0
```

尾流只在截止模式启用。它在比理论截止点更强的反向电压一侧，会按当前滤光片对应的截止电压自动调整补偿宽度，从截止模式最低电压附近平滑过渡到理论截止点；在理论截止点右侧也按一段平滑局部补偿曲线连续衰减。当前右侧衰减不是线性快速消失，而是让补偿在理论截止点右侧的局部范围内更慢释放，再平滑归零。这样基础收集率的连续变化始终会进入电流计算，五个滤光片不会共用同一段固定窄补偿区，电流表在接近零点前也不会因为补偿突然集中生效而大幅跳变。对于 546 nm、577 nm 这类理论截止电压较小的滤光片，这种更慢释放的右侧曲线可以避免补偿项下降过快、正好抵消正常阴极电流上升，从而减少电流接近 0 时“怎么调都不怎么动”的平台感。

### 6.4 伏安曲线的正向上升尺度

正向上升尺度 `Ur` 与电子最大初动能相关。当前框架认为：

```text
Ur 随 Kmax 改变
```

也就是说，频率和材料会影响伏安曲线达到饱和所需的正向电压。

但光强不进入 `Ur` 的计算。光强只改变电流纵轴尺度，不改变曲线在电压轴上的饱和位置。

### 6.5 95% 和 99% 饱和电压

项目中有两个常用指标：

```text
U95：收集率达到 95% 时的正向电压
U99：收集率达到 99% 时的正向电压
```

它们通过收集率公式反解得到：

```text
Uratio = Ur · [-ln((1 - ratio) / (1 - C0))]^(1 / b)
```

重要区别：

- `Vs` 是反向截止电压。
- `U95` 和 `U99` 是正向饱和电压指标。
- 这三者不是同一个物理量。

当前项目中，`U95` 和 `U99` 没有人为最大上限。只要频率、材料和模型参数导致它们变化，显示就应跟随变化。

## 7. 真实阴极电流

真实阴极电流指正常光电效应产生的阴极光电流。它只描述阴极因吸收光子而发射电子形成的电流。

计算思路为：

```text
I阴极 = Is · C有效(U)
```

其中：

- `Is` 是当前光通量下的饱和电流。
- `C有效(U)` 是当前外加电压下实际参与仪器电流计算的有效收集率。

在伏安模式下：

```text
C有效 = C基础
```

在截止电压模式且阳极寄生开启时：

```text
C有效 = C基础 + C尾流
```

约束：

```text
I阴极 >= 0
```

也就是说，正常阴极光电流不允许为负。若在界面上出现负向电流，它不应被解释为“阴极发出了负电流”，而应来自其他误差项、寄生项或仪器基准项。

## 8. 阳极寄生电流

### 8.1 物理含义

阳极寄生电流用于模拟以下情况：

- 阳极表面存在污染或光敏杂质。
- 杂散光或漏光照到阳极。
- 阳极侧也可能产生少量电子。
- 在反向电压场景下，这些电子对电流计读数产生负向贡献。

这不是正常阴极光电流，而是仪器或器件缺陷带来的寄生项。

### 8.2 当前计算依据

当前项目把阳极寄生电流写成阴极等效光电流的缩小比例：

```text
I阳极寄生 =
  -(I等效正向 / K) · p · s
```

其中：

- `I等效正向`：在相同材料、频率和光通量下，取相同电压绝对值作为正向电压时的等效阴极电流。
- `K`：阴极受光与阳极杂散受光之间的比例系数，当前 3D/实验仪器默认取 6。
- `p`：阳极污染程度。
- `s`：阳极漏光比例。
- 负号表示它对正常测量方向形成反向贡献。

当前规则：

```text
U >= 0 时，阳极寄生电流为 0
U < 0 时，阳极寄生电流可能为负
```

这样处理的物理含义是：在正向收集状态下，阳极寄生项不作为负向误差出现；在反向遏止或截止附近，阳极侧寄生电子可能形成更明显的负向残余读数。

## 9. 暗电流和本底电流

当前项目把暗电流、本底电流与阳极寄生电流区分处理。

### 9.1 原理动画中的处理

原理动画在截止电压演示中可以开启：

- 暗电流。
- 本底电流。
- 阳极寄生电流。

此时测量电流为：

```text
I测量 =
  I真实阴极
  + I阳极寄生
  + I暗
  + I本底
```

在伏安特性演示中，暗电流和本底电流被置为 0，只保留真实阴极电流和阳极寄生电流。

### 9.2 3D 仪器中的处理

3D 仪器当前主要考虑阳极寄生电流。暗电流和本底电流影响较小，暂不计入 3D 仪器读数。

因此 3D 区当前物理测量电流为：

```text
I测量 =
  I真实阴极
  + I阳极寄生
```

这里的真实阴极电流包含一层基础实验波动，用来模拟光电流的自然读数扰动。

## 10. 测量电流与仪器显示电流

当前项目中必须区分两个量：

1. 物理测量电流。
2. 仪器屏幕显示电流。

### 10.1 物理测量电流

物理测量电流表示光电管物理过程合成后的电流：

```text
I测量 =
  I真实阴极
  + I阳极寄生
  + 其他当前启用的残余项
```

在 3D 仪器中，目前其他残余项为 0。

### 10.2 仪器显示电流

仪器显示电流还要考虑电流计自身基准：

```text
I显示 =
  I测量
  + I基准
```

其中 `I基准` 来自：

- 仪器初始零漂。
- 用户调零旋钮造成的补偿。

如果后方电流线没有接入，电流屏只显示基准电流：

```text
I显示 = I基准
```

如果后方电流线已经接入，电流屏显示：

```text
I显示 = I测量 + I基准
```

因此，3D 电流显示屏显示的是仪器读数，不是单独的真实阴极电流。

## 11. 3D 区电流的当前计算流程

3D 区电流可以按以下顺序理解：

1. 判断光电效应是否可发生：

```text
光源开启
光路打开
光强大于 0
入射光频率高于材料阈频
```

2. 根据频率和材料计算：

```text
E = hν
Kmax = max(0, hν - φ)
Vs = Kmax
```

3. 根据光阑和距离计算相对光通量：

```text
G = (D / D0)^2 · (d0 / d)^2
```

4. 根据相对光通量计算饱和电流：

```text
Is = Iref · η · G
```

必要时再考虑仪器上限的渐近限制。

5. 根据外加电压计算收集率：

```text
C有效(U)
```

6. 得到真实阴极电流：

```text
I真实阴极 = Is · C有效(U)
```

7. 加入小幅基础读数波动：

```text
I真实阴极读数 = I真实阴极 · (1 + 小幅波动)
```

8. 在反向电压且阳极污染开启时，计算阳极寄生电流：

```text
I阳极寄生 =
  -(I等效正向 / 6) · 污染比例 · 漏光比例
```

9. 得到 3D 物理测量电流：

```text
I测量 = I真实阴极读数 + I阳极寄生
```

10. 如果显示在 3D 仪器电流屏上，再叠加仪器基准：

```text
I显示 = I测量 + I基准
```

未接入电流线时：

```text
I显示 = I基准
```

## 12. 电流曲线区域的显示口径

当前电流曲线区域按演示模式区分。

### 12.1 截止电压模式

截止电压模式下，曲线区域显示：

| 行 | 显示内容 | 物理意义 |
|---|---|---|
| 第一行 | 真实阴极电流 | 判断正常光电流是否被截止 |
| 第二行 | 阳极电流 | 单独观察阳极寄生项的方向和量级 |
| 第三行 | 测量电流 | 包含寄生项和残余项后的电流 |

这样可以让协作者看到：即使真实阴极电流已经接近 0，测量电流仍可能因阳极寄生电流而出现负向残余；第二行直接暴露阳极寄生项，便于区分“阴极已截止”和“仪器读数仍受阳极侧影响”。

### 12.2 伏安特性模式

伏安特性模式下，曲线区域显示：

| 行 | 显示内容 | 物理意义 |
|---|---|---|
| 第一行 | 电流 | 当前测量电流 |
| 第二行 | 外加电压 | 当前扫描电压 |
| 第三行 | 收集率 | 当前电压下的电子收集比例 |

伏安模式重点是观察 `I-U` 曲线如何随电压从反向遏止走向正向饱和。

## 13. 实验记录与数据分析

实验记录中会保留两类数据：

1. 仪器读数。
2. 物理拆分量。

仪器读数用于模拟真实实验记录，会带有一次额外测量噪声。

物理拆分量用于分析和排查，包括：

- 正常阴极电流。
- 阳极寄生电流。
- 测量电流。
- 收集率。
- 饱和电流。
- 理论截止电压。
- 手动记录的截止电压。
- 截止电压偏差。
- 截止数据质量。

用于估算普朗克常量时，只采用可信的截止电压记录。当前逻辑要求：

- 记录处于截止电压测量模式。
- 基础截止门的收集率足够接近 0。
- 手动截止电压与理论截止电压偏差在允许范围内。
- 同一频率重复记录时，优先使用最新有效记录。

这里特别区分“基础截止门”和“有效尾流”：前者用于判断物理截止是否到达，后者用于在仪器读数中抵消阳极寄生造成的负向偏移。有效尾流不参与判断电子是否已经满足理论截止条件。

拟合关系为：

```text
Uc = (h / e)ν - φ / e
```

通过多组不同频率下的截止电压，可以线性拟合出普朗克常量和材料逸出功。

## 14. 原理动画与 3D 区的区别

### 14.1 共同点

二者都遵循：

```text
E = hν
Kmax = max(0, hν - φ)
Vs = Kmax
I阴极 = Is · C(U)
```

并且都把阳极寄生电流作为独立负向项处理。

### 14.2 不同点

原理动画是教学演示区：

- 可以单独调节参数。
- 可以演示暗电流和本底电流。
- 可用于解释低于阈频、遏止、截止、正向收集等现象。

3D 区是实验仪器区：

- 使用实验台当前状态。
- 使用光阑和距离计算真实相对通量。
- 使用仪器调零和量程显示。
- 当前只把阳极寄生电流作为主要残余电流来源。
- 暗电流和本底电流暂不计入仪器读数。

## 15. 当前模型的近似与限制

当前框架是教学和实验复现模型，不是完整电极几何求解或空间电荷仿真。

主要近似包括：

1. 收集率采用经验分段函数，而不是逐个电子真实求解。
2. 正向饱和过程用指数趋近描述。
3. 反向截止过程用最大初动能和经验衰减描述。
4. 阳极寄生电流用等效正向电流缩小比例表示。
5. 3D 粒子数量是可视化密度，不等于真实电子数。
6. 暗电流和本底电流目前只在原理动画中作为教学项显式演示。
7. 电流读数中有小幅随机波动，用于模拟实验读数的不稳定性。

这些近似的目标是让实验现象、数据趋势和教学解释相互一致。

## 16. 后续讨论物理模型时的建议

后续若要调整物理框架，建议优先明确以下问题：

1. 要调整的是截止电压、饱和电流、收集率、寄生电流，还是仪器读数？
2. 调整是否符合“光强不影响截止电压”的基本理论？
3. 是否把真实阴极电流和寄生电流混在了一起？
4. 是否把反向截止电压和正向饱和电压混为一谈？
5. 3D 仪器读数是否需要新增暗电流或本底电流？
6. 若新增残余电流，应如何定义它的方向、大小、适用电压范围和实验意义？
7. 若修改伏安曲线，应说明修改的是收集率函数还是饱和电流尺度。
8. 若修改阳极寄生项，应说明污染程度、漏光比例和阴极等效电流之间的新关系。

只要保持这些边界清晰，项目中的 2D 实验台、3D 仪器、原理动画、电流曲线和实验记录就能继续保持同一套物理解释。
