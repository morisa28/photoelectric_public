## 4. 下次怎么继续改

灯点半径定义在同一文件：

```js
const INDICATOR_DOT_RADIUS = 4
```

所以：

- 直径 = `8px`
- 上下移动 `1` 个直径 = `8px`
- 上下移动 `2` 个直径 = `16px`

该贴面画布高度是：

```js
const SCREEN_SURFACE_SIZE = {
  width: 1024,
  height: 512,
}
```

因此 `y` 坐标换算规则是：

- 上移 `8px`：`y - 0.015625`
- 上移 `16px`：`y - 0.03125`
- 上移 `100px`：`y - 0.1953125`
- 下移 `8px`：`y + 0.015625`
- 下移 `16px`：`y + 0.03125`
- 下移 `150px`：`y + 0.29296875`

如果下次只想改单个灯，就只改：

- `voltAmpere`
或
- `cutoff`

不要去改另一组：

```js
const WORK_MODE_INDICATOR_POINTS
```

那一组是“手动 / 自动”工作模式灯，不是这次改的对象。
