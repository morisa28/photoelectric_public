# 阶段文档22-22-new(26)-截止模式阴极尾流A路径

## 1. 阶段目标

根据用户要求，先提交当前基线后尝试 A 路径：在不修改阳极寄生电流生成逻辑、不新增特殊显示标准、不影响伏安模式的前提下，让截止电压模式中按测量电流归零时的零点回到理论截止电压附近。

本阶段重点处理 365 nm、Cs、默认光阑和默认距离下，理论截止电压约为 `1.447 V`，但此前测量电流零点会被阳极寄生拉偏的问题。

## 2. 实现内容

1. 模型层新增“基础收集率”和“截止尾流”两层口径。
   - 基础收集率仍保持非包围式光电管模型。
   - 在 `U <= -Vs` 时，基础收集率仍关闭为 0。
   - 有效收集率用于实际阴极电流计算。

2. 截止模式新增局部有效阴极尾流。
   - 尾流只在截止电压模式启用。
   - 尾流强度按当前阳极寄生参数在 `U = -Vs` 处的负向电流反推。
   - 目标是让 `I_effective_cathode + I_anode_parasitic` 在理论截止点附近接近 0。

3. 伏安模式保持原逻辑。
   - 伏安模式不启用截止尾流。
   - 阳极寄生电流仍使用反向电压下的正向等效阴极电流缩放公式。

4. 截止质量判定保持看基础截止门。
   - 记录中新增基础收集率、截止门收集率、尾流收集率字段。
   - `cutoffQuality` 使用基础截止门，不把有效尾流误判为“尚未截止”。

## 3. 关键数值

默认条件：

```text
阴极材料：Cs
滤光片：365 nm
光阑：4 mm
距离：40 cm
阳极寄生默认：K = 6，污染 0.52，漏光 0.58
```

在 `U = -Vs ≈ -1.447 V` 附近：

```text

阳极寄生电流 ≈ -8.06e-13 A
物理测量电流 ≈ 0 A
```

在 `U = -1.998 V`：

```text
有效阴极尾流已衰减为 0

物理测量电流 ≈ -1.01e-12 A
1e-13 A 档电流屏读数 ≈ -10.1
```

## 6. 风险与后续观察

- A 路径本质上是截止模式的有效读数补偿，不是修改基础光电效应截止公式。
- 若后续认为“真实阴极电流”必须严格只表示基础阴极光电流，则需要在 UI 文案中进一步区分“基础阴极电流”和“有效阴极读数”。
- 当前实现优先满足用户指定约束：不改阳极电流、不新增特殊显示标准、先让截止模式测量零点回到理论截止电压附近。
