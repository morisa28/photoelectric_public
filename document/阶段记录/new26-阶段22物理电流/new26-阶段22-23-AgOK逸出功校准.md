# 阶段文档22-23-new(26)-AgOK逸出功校准

## 1. 阶段目标

根据用户要求，将 Ag-O-K 复合光电阴极的逸出功从 `1.55 eV` 调整为 `1.65 eV`。

这次调整的目的，是让 Ag-O-K 可选材料更接近用户提供的实验表格。该表格按光电效应方程反推的等效逸出功约为 `1.64 eV ~ 1.66 eV`。

## 2. 修改内容

1. 更新材料表：

```text
Ag-O-K: φ = 1.65 eV
```

2. 更新当前物理框架说明文档中的 Ag-O-K 逸出功表。

3. 更新物理模型测试中的 Ag-O-K 基准逸出功，避免测试继续使用旧的 `1.55 eV`。

4. 同步更新原始物理修改指导文档中的材料表示例，避免后续按旧参数再次实施。

## 3. 理论截止电压影响

按：

```text
U0 = hν - φ
```

取 `φ = 1.65 eV` 时，用户表格对应频率下的理论截止电压约为：

| 波长 | 频率 | Ag-O-K φ=1.65 eV 理论值 |
|---:|---:|---:|
| 365.0 nm | 8.219e14 Hz | 1.749 V |
| 404.7 nm | 7.413e14 Hz | 1.416 V |
| 435.8 nm | 6.884e14 Hz | 1.197 V |
| 546.1 nm | 5.493e14 Hz | 0.622 V |
| 577.0 nm | 5.199e14 Hz | 0.500 V |

这组值与用户提供表格中的 `1.758 V、1.422 V、1.191 V、0.625 V、0.491 V` 已处于同一近似区间。

## 4. 影响范围

- 当前默认阴极材料仍为 Cs，不改默认启动状态。
- 3D 区和 2D 实验台仍共用主实验状态；当用户切换到 Ag-O-K 时，3D 区会使用新的 `1.65 eV`。
- 该修改只改变 Ag-O-K 对应的理论截止电压、阈频和相关电流模型结果。
- Cs、K、Na、Zn、Cu 等其他材料参数不变。
- 截止模式 A 路径中的阳极寄生与阴极尾流逻辑不变。
