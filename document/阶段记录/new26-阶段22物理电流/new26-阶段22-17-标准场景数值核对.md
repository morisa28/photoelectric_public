# 阶段文档22-17-new(26)-标准场景数值核对

## 阶段目标

按 `当前代码适配版电流逻辑实施方案-new(26).md` 的 P0 要求，对当前模型做 Cs 五个滤光片标准场景数值核对，确认截止电压偏低现象的来源，为后续只处理截止模式记录/质量口径提供依据。

## 完成内容

- 新增 `document/当前总纲/04-当前模型标准场景数值核对表-new26.md`。
- 核对标准场景：
  - Cs 阴极。
  - 365 / 405 / 436 / 546 / 577 nm。
  - 光阑 4 mm。
  - 光电管距离 40 cm。
  - 截止电压模式。
  - 保留当前默认阳极寄生。
  - 暗电流、本底电流不进入 3D 仪器读数。
- 输出三组表：
  - 理论截止点附近的真实阴极电流、阳极寄生电流和物理测量电流。
  - 默认 `-1.998 V` 下的电流分量。
  - 若错误地按 `I_measured = 0` 求表观截止点时得到的偏低电压。
- 结论：
  - 当前阴极主光电流在理论截止点已接近 0 或为 0。
  - 偏低数据来源于真实阴极电流与阳极寄生电流提前抵消。
  - 后续不应回退电流模型，而应处理截止记录质量和已有显示口径。
