# 阶段文档22-26-new(26)-截止尾流平滑补偿曲线

## 1. 阶段目标

根据用户反馈，3D 区实验在电流表示数接近 0 时，调节电压对电流变化的影响会突然变小。原因是上一阶段 A 路径中，截止点右侧使用了固定尾流平台：

```text
C_effective = max(C_base, C_tail)
```

在基础收集率尚未超过尾流前，基础曲线的连续变化会被平台遮住。本阶段将其改为更平滑的局部补偿曲线。

## 2. 修改内容

1. 尾流曲线改为 smoothstep 局部曲线。
   - `U < -Vs` 一侧：在约 `0.12 V` 范围内平滑衰减。
   - `U >= -Vs` 一侧：不再固定为平台，而是按动态估算宽度平滑衰减。

2. 有效收集率从取最大值改为加性叠加：

```text
C_effective = C_base + C_tail
```

这样基础收集率的连续变化始终参与电流计算。

3. 右侧补偿宽度动态估算。
   - 先求“无尾流时基础阴极电流自然抵消阳极寄生”的电压位置。
   - 再按比例扩展为右侧补偿曲线宽度。
   - 目的：避免补偿太早衰减导致第二个零点，也避免固定平台导致零点附近电流响应迟钝。

## 3. 默认场景数值

默认条件：

```text
阴极材料：Ag-O-K
滤光片：365 nm
光阑：4 mm
距离：40 cm
阳极寄生默认：K = 6，污染 0.52，漏光 0.58
```

计算结果：

```text
理论截止电压 Vs ≈ 1.7470 V
右侧补偿宽度 ≈ 0.8856 V
理论截止点尾流收集率 ≈ 0.011834
```

电压调节时的测量电流趋势：

| 外加电压 | 测量电流，按 1e-13 A 档折算 |
|---:|---:|
| -1.997 V | -11.317 |
| -1.747 V | 0.000 |
| -1.697 V | 0.186 |
| -1.647 V | 0.306 |

相比固定平台，新曲线在零点右侧会让基础收集率和尾流衰减同时进入读数，电流表不再由一个固定尾流下限主导。

## 4. 影响范围

- 只影响截止电压模式下的阴极尾流补偿形状。
- 不修改阳极寄生电流生成逻辑。
- 不修改伏安模式。
- 不修改材料参数、光阑、距离、滤光片或量程。
- 截止质量判定仍使用基础截止门收集率，不使用尾流后的有效收集率。
