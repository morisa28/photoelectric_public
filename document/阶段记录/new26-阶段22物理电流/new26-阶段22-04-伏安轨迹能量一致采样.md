# 阶段文档22-4-new(26)-伏安轨迹能量一致采样

## 阶段目标

按照修改指南 Phase 4，弱化旧伏安轨迹中的经验电压比例和大于 1 的速度倍率，让伏安轨迹采样与 `Kmax`、出射角、外加电压和极板距离保持一致。

## 完成内容

- `src/features/photoelectric-animation/voltAmpTrajectory.js`
  - 删除 `VOLT_AMP_INITIAL_SPEED_SCALE = 72` 和 `VOLT_AMP_ACCELERATION_SCALE = 4200` 的经验组合。
  - 使用阶段 1 的 `resolveEnergyConsistentMotion()` 计算 `vx`、`vy` 与 `accelerationPxPerS2`。
  - 将速度倍率采样替换为动能因子采样：
    - `0.08`
    - `0.18`
    - `0.35`
    - `0.60`
    - `0.85`
    - `1.00`
  - 每个样本新增 `sampleKineticEnergyEv`、`longitudinalKineticEnergyEv` 和 `energyFactor`。
  - 反向电压下先检查 `Kx + V > 0`，不满足时标记为 `returned-to-cathode`。
- `src/features/photoelectric-animation/physics.js`
  - 将 `accelerationMagnitudeProxy` 改名为 `animationAccelerationProxy`，避免把动画比例量误解为真实 SI 加速度。
- `src/features/photoelectric-animation/engine.js`
  - 光电子初速度倍率从 `1–2` 改为 `0.25–1`，避免动画电子动能超过 `Kmax`。
  - 正常电子事件新增调试字段：
    - `speedMultiplier`
    - `sampleKineticEnergyEv`
    - `emissionAngleRadians`
    - `longitudinalKineticEnergyEv`
  - 调试快照同步输出这些字段，方便后续检查电子能量是否超过 `Kmax`。
- 测试同步更新：
  - 伏安轨迹测试改为能量一致采样口径。
  - physics 测试检查 `animationAccelerationProxy`。
  - engine 测试更新速度倍率区间，并调整保持返回/飞出场区行为的夹具初速。
  - mapping 测试更新能量一致 motion 的加速度主导比例和速度关系断言。

## 物理行为变化

- 伏安轨迹样本的动能不再超过 `Kmax = hν - φ`。
- 反向电压下，电子是否能穿过遏止场由纵向动能 `Kx` 决定。
- 0 V 和低正压下不再依赖旧速度倍率制造过度收集。
- 动画加速度字段不再以真实 SI 加速度名义暴露。

## 本阶段边界

- 伏安模式宏观收集率仍由 Phase 3 接入的统一非包围式模型决定。
- 轨迹采样器的 `collectionRatio` 主要用于诊断和 motion；真实动画状态的收集率以 `resolveCathodePhotocurrentAmp()` 为准。
- 记录和报告字段尚未扩展，后续 Phase 5 处理。
