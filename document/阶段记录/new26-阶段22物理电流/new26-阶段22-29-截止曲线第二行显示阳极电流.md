# 阶段文档22-29-new(26)-截止曲线第二行显示阳极电流

## 1. 阶段目标

用户要求：截止电压模式下，“电流曲线”区域第二行由电压改为显示阳极电流。

本阶段只调整显示口径：

1. 不修改理论截止电压。
2. 不修改阴极电流、阳极寄生电流或测量电流计算。
3. 不修改伏安模式的三行指标。
4. 不修改仪器电流屏显示口径。

## 2. 修改内容

截止电压模式下，电流曲线三行指标调整为：

| 行 | 修改后显示 |
|---|---|
| 第一行 | 真实电流 |
| 第二行 | 阳极电流 |
| 第三行 | 测量电流 |

其中：

- “真实电流”对应真实阴极电流。
- “阳极电流”对应阳极寄生电流，保留正负号显示；当前默认反向电压下通常为负值。
- “测量电流”仍为真实阴极电流与阳极寄生电流等物理项合成后的测量口径。

伏安特性模式保持原有显示：

| 行 | 显示内容 |
|---|---|
| 第一行 | 电流 |
| 第二行 | 电压 |
| 第三行 | 收集率 |

## 3. 视觉与布局处理

“阳极电流”比原来的“电压”标签更长，因此本阶段同步加宽电流曲线统计行的标签列，避免标签换行或挤压条形区域。

备用的画布内统计条也同步改为截止模式显示阳极电流，避免后续重新启用画布内曲线时口径回退。

## 4. 影响范围

- 影响原理动画/3D 微观小窗复用的电流曲线统计区域。
- 影响截止电压模式下第二行显示内容。
- 不影响伏安模式。
- 不影响 store 中的电流计算。
- 不影响实验记录、拟合、调零和仪器显示电流。
