# 阶段文档22-6-new(26)-电流曲线读数对齐底层模型

## 阶段目标

修正原理动画/3D 微观小窗中“电流曲线”区域的显示口径，让右上角电流值、条形长度和收集率与底层物理模型计算结果保持一致，不再被动画引擎实时粒子统计的随机窗口影响。

## 完成内容

- 修改 `src/components/animation/PhotoelectricAnimationPanel.vue`。
- “电流曲线”区域的电流显示改为读取 `animationState.measuredPhotocurrentAmp` / `animationState.photocurrentAmp`。
- 电流条长度改为读取 `animationState.measuredCurrentSignedLevel`。
- 收集率显示和条形长度改为读取 `animationState.collectorSuccessRatio`。
- 迷你电流曲线改为以底层模型电流等级绘制稳定曲线，避免与右侧电流读数出现不一致。
- 保留 Canvas 引擎实时统计的接收逻辑，但不再用它覆盖 DOM 读数。
- 新增组件测试：即使 Canvas/动画引擎上报 0 电流、0 收集率，DOM 面板仍显示底层模型电流和收集率。
