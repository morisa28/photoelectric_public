# 阶段文档22-5-new(26)-记录字段与截止质量拟合

## 阶段目标

按照修改指南 Phase 5，扩展实验记录中的物理模型字段，并给截止电压数据增加质量标记，使普朗克常量拟合不再直接使用任意负电压记录。

## 完成内容

- `src/stores/experiment.js`
  - `recordData()` 新增模型字段：
    - `normalCathodePhotocurrentAmp`
    - `anodeParasiticPhotocurrentAmp`
    - `darkCurrentAmp`
    - `backgroundCurrentAmp`
    - `measuredPhotocurrentAmp`
    - `collectionFraction`
    - `saturationCurrentAmp`
    - `voltAmpRiseVoltage`
    - `saturationVoltage95`
    - `saturationVoltage99`
    - `cutoffVoltageSource`
    - `cutoffQuality`
  - 新增截止电压质量判定：
    - `valid`：净阴极收集率接近 0 且与理论截止电压偏差较小。
    - `suspect`：接近零电流或偏差中等，但不满足有效拟合条件。
    - `invalid`：不满足截止数据要求。
  - 伏安模式记录的 `cutoffQuality` 为 `null`，不参与截止电压拟合。
- `src/features/experiment/helpers.js`
  - `buildFitRecords()` 改为只使用 `cutoffQuality === 'valid'` 的截止记录。
  - 对没有 `cutoffQuality` 的历史旧记录按兼容有效处理，避免旧数据突然失效。
- `src/stores/experiment.spec.js`
  - 新增记录字段测试。
  - 新增 tuned cutoff 数据标记为 `valid` 的测试。
  - 新增 `suspect` 截止记录不进入 `analysisSummary.fitRecords` 的测试。

## 物理与数据处理影响

- 手动截止电压记录不再天然等同于有效拟合点。
- 拟合只使用质量为 `valid` 的截止记录，降低错误零电流点对 `h` 和逸出功估算的影响。
- 记录中保留模型派生的收集率、饱和电流和 U95/U99，后续可以用于报告解释或伏安曲线对照。

## 本阶段边界

- UI 仍主要展示原始历史记录，质量字段尚未做专门筛选视图。
- 阳极寄生、暗电流、本底电流在真实 store 记录中先保留为可扩展字段；当前主 store 读数仍以正常阴极电流和仪器基准电流为主。
- 旧记录兼容策略保留，后续如需要严格数据治理，可在迁移时显式标记旧记录质量。
