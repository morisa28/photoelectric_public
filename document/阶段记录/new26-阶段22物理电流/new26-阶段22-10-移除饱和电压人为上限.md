# 阶段文档22-10-new(26)-移除饱和电压人为上限

## 阶段目标

按真实光电效应物理口径修正伏安模式的 U95/U99 计算：移除饱和电压尺度的人为上限，并避免光强改变饱和电压。

## 完成内容

- `src/features/experiment/photoelectricModel.js`
  - `resolveVoltAmpRiseVoltage` 不再接收或应用 `maxRiseVoltage`。
  - 移除光强通量对 `riseVoltage` 的影响。
  - `riseVoltage` 只随频率、材料功函数和模型几何尺度变化。
  - 保留 `minRiseVoltage` 作为近阈值/低动能情况下的最小稳定尺度，避免曲线尺度退化为 0。
- `src/features/experiment/constants.js`
  - 从 `NON_ENCLOSING_PHOTOTUBE_MODEL` 移除 `maxRiseVoltage`。
  - 从 `NON_ENCLOSING_PHOTOTUBE_MODEL` 移除 `intensityCoefficient`。
- `src/features/experiment/photoelectricModel.test.js`
  - 新增测试：光强变化时 U95/U99 不变。
  - 新增测试：高频情况下 U95/U99 不被 8 V 上限钳制。
- `document/photoelectric_codex_modification_guide.md`
  - 同步规范：光强只影响饱和电流，不影响 U95/U99。
  - 同步规范：不设置人为最大饱和电压上限。

## 验证结果

```bash
npm run test:unit -- --run src/features/experiment/photoelectricModel.test.js src/stores/experiment.spec.js src/components/__tests__/PrincipleAnimationSection.spec.js
```

结果：

- 相关测试：3 个测试文件、26 个测试全部通过。

页面验证：

- 使用 Playwright 检查 `http://127.0.0.1:5173/` 的“原理动画 / 伏安特性”入口。
- 同一频率 `8.214e14 Hz` 下：
  - 光强 `10%`：U95 = `22.9 V`，U99 = `35.5 V`。
  - 光强 `100%`：U95 = `22.9 V`，U99 = `35.5 V`。
  - 真实阴极电流随光强从 `2.39e-13 A` 增至 `1.87e-12 A`。
- 光强 `100%` 下：
  - 频率 `5.49e14 Hz`：U95 = `16.5 V`，U99 = `25.5 V`。
  - 频率 `8.3e14 Hz`：U95 = `23.1 V`，U99 = `35.7 V`。

## 影响范围

- 影响伏安模式中 U95/U99、`voltAmpRiseVoltage`、收集率曲线尺度和依赖该模型的记录字段。
- 光强仍会改变饱和电流和测量电流，不再改变 U95/U99。
- 不影响理论截止电压公式 `Vs = hν - φ`。
