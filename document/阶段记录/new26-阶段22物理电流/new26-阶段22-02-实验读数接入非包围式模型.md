# 阶段文档22-2-new(26)-实验读数接入非包围式模型

## 阶段目标

按照修改指南 Phase 2，将阶段 1 的统一物理内核接入真实实验 store，使 2D/3D 仪表读数从包围式 `tanh` 近似切换到非包围式光电管模型。

## 完成内容

- `src/features/experiment/constants.js`
  - 新增 `agOK` 复合光电阴极，功函数为 `1.55 eV`。
  - 新增 `NON_ENCLOSING_PHOTOTUBE_MODEL`，包含 0V 收集率、正向饱和尺度、频率/强度修正和 UI 饱和比例参数。
  - 新增 `PHOTOCURRENT_PRESETS`，包含归一化教学演示和 `ZKY-GD-3` 论文数据近似两档。
- `src/features/experiment/helpers.js`
  - 默认材料从 `cesium` 改为 `agOK`。
  - 默认仪器状态新增 `photocurrentPreset: 'zkyGd3Like'`。
- `src/stores/experiment.js`
  - 新增 `physicalLightFluxRatio` getter，按光阑面积和距离平方计算真实物理光通量。
  - 新增 `cathodePhotocurrentModel` getter，集中返回阴极电流、收集率、饱和电流、U95/U99 和理论截止电压。
  - 用 `resolveCathodePhotocurrentAmp()` 替换 `generatedPhotocurrentAmp()` 中的 `tanh((V + cutoff) / 0.45)`。
  - 正常阴极光电流强制保持非负。
  - `recalculateIntensity()` 改为光阑面积比例；`params.intensity` 仍只作为 UI 百分比显示，8 mm 光阑真实通量不会被 100% UI 上限截断。
- 新增 `src/stores/experiment.spec.js`，覆盖默认材料/预设、光阑面积比例、0V 不饱和、30V 参考近饱和和非负阴极电流。

## 物理行为变化

- 真实电流不再在 0 V 附近直接饱和。
- 546 nm、Ag-O-K、4 mm、40 cm 参考条件下，0 V 收集率约 5%，30 V 接近 99% 收集。
- 2/4/8 mm 光阑真实光通量比例为 1:4:16。
- 饱和电流随真实光通量增加。
- 频率升高时，模型允许 U95 上升，不再强行固定为低压饱和。
