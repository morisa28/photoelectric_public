# new(7) 阶段文档 4

## 本轮定位

本轮是基于阶段文档 3 的自我迭代，不再继续大拆结构，只做边界清理。

目标：

1. 去掉首轮拆分后残留的重复常量
2. 让 composable 更少依赖 store 包装层
3. 再次验证构建

## 本轮完成内容

### 1. 收紧 2D 控制器依赖

`src/composables/useVirtualLabController.js` 现在直接依赖：

1. `src/features/experiment/constants.js`
2. `src/features/experiment/helpers.js`

而不是再通过 `store` 文件中转读取这些静态内容。

这样做的作用是：

1. 明确它依赖的是配置与纯工具
2. 减少 UI 控制器对 store 包装层的耦合

### 2. 去掉 3D 控制器中的重复量程数组

`src/composables/usePhotoelectricSceneController.js` 把本地写死的 `CURRENT_RANGES` 改为直接引用实验常量层。

结果：

1. 不再重复定义量程列表
2. 避免未来只改一处导致 2D / 3D 行为漂移
