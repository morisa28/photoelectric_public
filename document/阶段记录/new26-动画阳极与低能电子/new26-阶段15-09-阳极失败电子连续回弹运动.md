# 阶段文档15-9-new(26)-阳极失败电子连续回弹运动

## 本阶段目标

- 修正阳极失败电子上一阶段的速度突变问题。
- 阳极失败电子应先向阴极方向行进一段距离，再在速度降为 0 后反向回弹，并在回弹结束前再次减速到 0 后淡出。

## 实现内容

- 保留上一阶段的前进距离约束：
  - 前进距离仍按完整距离的 25% 到 80% 之间采样。
  - 反作用电场越强，前进距离上限越小，最低不低于 25%。
- 前向受阻段改为解析式匀变速运动：
  - 以阳极电子初始速度和预定前进距离计算恒定反向加速度。
  - 电子向阴极方向运动时逐渐减速。
  - 到达转向点时 `vx = 0`、`vy = 0`，不再发生方向突变。
- 反向回弹段改为三角速度曲线：
  - 从转向点静止开始。
  - 向阳极方向先均匀加速，再均匀减速。
  - 淡出终点速度回到 0。
- 绘制透明度改为跟随实际回弹持续时间，避免运动持续时间和淡出节奏脱节。
- 调试快照补充：
  - `vy`
  - `retardedForwardAge`
  - `retardedForwardDuration`
  - `retardedReverseAge`
  - `retardedReverseDuration`
  - `retardedTurnX`

## 测试覆盖

- 更新阳极污染电子集成测试：
  - 电子在转向点进入 `retardedFading` 时，`vx` 与 `vy` 均为 0。
  - 转向点仍满足前进距离比例约束。
  - 失败过程不计入阴极接收，不触发阴极接收闪光。
  - 回弹段逐帧推进，验证：
    - 回弹开始后 `vx > 0`。
    - 回弹中段速度不小于开始段速度。
    - 回弹结束时 `vx` 回到 0。
    - 回弹过程 `x` 单调向阳极方向增加。
