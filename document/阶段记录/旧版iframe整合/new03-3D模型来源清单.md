# new（3）3D模型来源清单

## 1. 说明

本清单用于记录光电效应实验平台 3D 器材的在线候选模型来源，便于后续将当前程序化 3D 场景替换为真实外部模型资产。

当前 `new（3）` 已在页面中内置“模型来源候选”面板，并同步展示以下资源。

说明：

1. 公开模型站点中“光电效应专用汞灯 / 光电管 / 试验仪”精确命名资源较少
2. 因此部分资源采用“结构近似替代”的方式先完成器材占位与语义映射
3. 后续若拿到更高拟真度或更符合实验教学规范的模型，可直接替换

## 2. 候选资源

### 汞灯

- 候选模型：Light Tube
- 来源：Sketchfab
- 匹配方式：近似替代
- 说明：先以管状灯具模型匹配汞灯几何外形与发光区域，再由页面逻辑补充“盖子”“电源”“光束”实验语义
- 链接：https://sketchfab.com/3d-models/light-tube-12e2e5a067c14c928f883abd2f75fb94

### 光电管

- 候选模型：Vacuum Tube
- 来源：Sketchfab
- 匹配方式：近似替代
- 说明：公开资源中精确“光电管”较难直接检索，当前用真空电子管模型承接玻璃外壳与电极结构
- 链接：https://sketchfab.com/3d-models/vacuum-tube-950f88dd603e4b178cebc0a550d7fa06

### 光阑

- 候选模型：Iris Mechanism Generator (Variant 4)
- 来源：GrabCAD
- 匹配方式：结构可复用
- 说明：可调光阑机构与实验所需孔径控制高度一致，适合转为 Φ2 / Φ4 / Φ8 三档可视化组件
- 链接：https://grabcad.com/tutorials/iris-mechanism-generator-variant-4

### 试验仪

- 候选模型：Multimeter - FREE
- 来源：Sketchfab
- 匹配方式：近似替代
- 说明：用于承接电压 / 电流读数显示的仪表外观，页面内仍以实验数据逻辑为准
- 链接：https://sketchfab.com/3d-models/multimeter-free-8f0f25071ae84d8d9cce0cd1cc27a2bf

### 线材

- 候选模型：Cables
- 来源：Sketchfab
- 匹配方式：可直接复用
- 说明：适合替换当前程序化生成的高频输入线与连接线
- 链接：https://sketchfab.com/3d-models/cables-a4d0d3d9fe4d4f33b5fe975d6794b92a

## 3. 当前实现方式

当前 `new（3）` 没有直接依赖外网模型下载，而是先采用以下方案：

1. 页面内使用程序化 3D 场景保证离线可运行和构建稳定
2. 用统一状态源把器材状态、实验参数、历史记录与 2D 页面同步
3. 在 3D 页面显式展示候选模型来源，后续可按器材逐一替换
4. 候选模型清单已同步写入 store 派生快照，2D / 3D 可共享同一份来源数据

## 4. 替换建议

后续如果继续迭代 3D 拟真度，建议顺序如下：

1. 优先替换汞灯、光电管与试验仪这三个辨识度最高的器材
2. 保持当前 DOM id 与状态映射关系不变，避免破坏 2D/3D 同步
3. 先做几何替换，再补材质、贴图、灯光和开盖动画
