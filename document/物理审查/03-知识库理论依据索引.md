## 可作为审查依据的理论条目

### K001 光子能量与频率/波长关系

- 理论名称：光子能量关系。
- 简要说明：单个光子的能量由频率决定，波长通过光速与频率互换。
- 对应公式：`E = h * nu = h * c / lambda`。
- 适用条件：同一光源谱线或滤光片近似单色光；若考虑滤光片带宽，需要另行复核原文或实验标定。
- 与项目仿真的关联：频率、波长和滤光片状态应决定能量条、阈频判断、最大初动能和理论截止电压。
- 后续审查要检查的实现点：`constants.js` 中波长/频率常量是否一致；`helpers.js`、`physics.js`、`mapping.js` 是否把能量主要绑定到频率/波长，而不是光强。
- 来源文件路径：`knowledge/claims.md` C001；`knowledge/topics/T01-frequency-wavelength-cutoff-voltage.md`；`knowledge/project-mapping.md`。
- 证据等级：A。
- 是否需要论文原文复核：否；若要核对具体谱线/滤光片带宽，则是。

### K002 爱因斯坦光电效应方程

- 理论名称：爱因斯坦光电效应方程。
- 简要说明：光电子最大初动能等于光子能量减去材料逸出功；低于阈频时不发生由该入射光触发的光电子发射。
- 对应公式：`K_max = h * nu - W0`，当 `h * nu <= W0` 时无光电发射或 `K_max` 不应为正。
- 适用条件：理想化单光子能量模型；不包含接触电势差、空间电荷、表面状态分布等高阶修正。
- 与项目仿真的关联：材料逸出功、频率和阈频应决定是否发射、势垒高度、最大初动能和动画阶段文案。
- 后续审查要检查的实现点：`physics.js`、`mapping.js`、`experiment.js` 是否正确处理低于阈频场景；动画是否避免在无光电效应条件下展示大量确定性光电子。
- 来源文件路径：`knowledge/claims.md` C001；`knowledge/topics/T01-frequency-wavelength-cutoff-voltage.md`；`knowledge/papers/P032.md`。
- 证据等级：A。
- 是否需要论文原文复核：否；若要处理材料表面修正，则是。

### K003 最大初动能与遏止/截止电压关系

- 理论名称：遏止电压关系。
- 简要说明：遏止电压的大小反映最大初动能；项目实现时需要特别注意电压正负号约定。
- 对应公式：`e * U_c = K_max`，等价于 `U_c = (h * nu - W0) / e`。若 UI 使用反向电压为负值，应比较绝对值或显式处理符号。
- 适用条件：理想截止电压；真实测量可能受本底、暗电流、阳极寄生项、读数阈值和外推方式影响。
- 与项目仿真的关联：截止电压测试、伏安曲线截止点、实验报告拟合和微观动画势垒都应共享同一物理口径。
- 后续审查要检查的实现点：`helpers.js`、`experiment.js`、`voltAmpTrajectory.js`、`IntegratedPlotPanel.vue` 是否在符号、单位和量程上保持一致。
- 来源文件路径：`knowledge/claims.md` C001；`knowledge/topics/T01-frequency-wavelength-cutoff-voltage.md`；`knowledge/topics/T03-volt-ampere-curve.md`。
- 证据等级：A。
- 是否需要论文原文复核：否；若要比较真实曲线外推值，则是。

### K004 截止频率与材料逸出功关系

- 理论名称：截止频率/阈频关系。
- 简要说明：材料逸出功越大，触发光电效应所需频率越高；低于阈频时增加光强不能补偿单光子能量不足。
- 对应公式：`nu_0 = W0 / h`，`lambda_0 = h * c / W0`。
- 适用条件：材料逸出功采用项目预设常量；真实材料表面状态、污染和接触电势差未纳入时只能作为教学模型。
- 与项目仿真的关联：材料切换应改变阈频、截止电压、电子发射条件和讲解文案。
- 后续审查要检查的实现点：`constants.js` 材料逸出功是否单位明确；`mapping.js` 是否把阈频判断用于发射率、能量条和状态文案。
- 来源文件路径：`knowledge/claims.md` C001；`knowledge/topics/T01-frequency-wavelength-cutoff-voltage.md`。
- 证据等级：A。
- 是否需要论文原文复核：否；若要验证具体材料数值，则是。

### K005 频率与截止电压线性关系

- 理论名称：`U_c - nu` 线性关系。
- 简要说明：固定材料时，截止电压随频率近似线性变化，线性拟合斜率可用于估计普朗克常量。
- 对应公式：`U_c = (h / e) * nu - W0 / e`。
- 适用条件：固定材料、有效单色光、测量值已处理系统误差；真实实验常通过外推、曲率法或线性回归降低误差。
- 与项目仿真的关联：数据记录、拟合曲线、普朗克常量估计和报告摘要都应基于多点数据，而不是单点读数。
- 后续审查要检查的实现点：`helpers.js` 的回归和普朗克常量估计；`experiment.js` 的记录字段；`dataDiagnostic.js` 的趋势诊断；`ExperimentReport.vue` 的误差说明。
- 来源文件路径：`knowledge/claims.md` C001、C007；`knowledge/topics/T01-frequency-wavelength-cutoff-voltage.md`；`knowledge/topics/T05-instrument-error-and-data-processing.md`。
- 证据等级：A。
- 是否需要论文原文复核：否；若要对照特定论文的数据处理细节，则是。

### K006 光强与光电流/饱和电流关系

- 理论名称：光强影响光子通量与饱和光电流。
- 简要说明：在频率高于阈频且材料、几何条件相同的情况下，光强主要改变单位时间入射光子数和逸出电子数，进而改变光电流与饱和平台。
- 对应规律：`photonRate ∝ intensity`，理想条件下 `I_s ∝ intensity`；单电子最大初动能仍由频率和逸出功决定。
- 适用条件：同一频率/材料/光路几何；过低光强、探测阈值、暗电流和光阑/距离会影响可观测电流。
- 与项目仿真的关联：光强滑块应主要影响 `photonRate`、发射密度、统计电流、饱和电流平台，而不是能量条或截止电压主项。
- 后续审查要检查的实现点：`experiment.js`、`mapping.js`、`PhotoelectricAnimationPanel.vue` 是否避免“光强越大电子能量越大”的错误表达；伏安曲线饱和平台是否随光强变动。
- 来源文件路径：`knowledge/claims.md` C002；`knowledge/topics/T02-light-intensity-saturation-current.md`；`knowledge/papers/P009.md`；`knowledge/papers/P019.md`；`knowledge/papers/P025.md`。
- 证据等级：A。
- 是否需要论文原文复核：需要定量比例或实验图表时复核。

### K007 光电流-电压曲线分区与连续性

- 理论名称：光电效应伏安特性曲线。
- 简要说明：伏安特性曲线应包含反向减速/截止附近、零偏压附近、加速收集和饱和平台等区段；真实或教学化曲线都不宜表现成硬跳变。
- 对应规律：反向电压增大时收集率降低并趋近截止；零偏压仍可能有部分电子到达阳极；正向加速电压提高收集率并最终接近饱和平台。
- 适用条件：光强、频率、材料和几何条件固定；曲线细节受初速度分布、空间电荷、阳极光电流、本底电流和仪器阈值影响。
- 与项目仿真的关联：伏安轨迹、收集率、飞行时间、电流曲线和讲解区段应保持连续且可解释。
- 后续审查要检查的实现点：`voltAmpTrajectory.js` 的采样函数；`engine.js` 的电子收集/回退；`IntegratedPlotPanel.vue` 的曲线绘制；`mapping.js` 的阶段文案。
- 来源文件路径：`knowledge/claims.md` C003；`knowledge/topics/T03-volt-ampere-curve.md`；`knowledge/papers/P025.md`；`knowledge/papers/P030.md`；`knowledge/papers/P032.md`。
- 证据等级：A；其中 P030 为 CAJ 元数据级，不能单独作为强证据。
- 是否需要论文原文复核：曲线细节和参数需要复核。

### K008 饱和电流概念

- 理论名称：饱和光电流。
- 简要说明：当电压足以收集绝大多数被释放的光电子时，光电流接近平台；平台高度主要受光强、量子效率和几何收集条件影响。
- 对应规律：`I_s` 近似随有效入射光强增大；继续提高正向电压时电流增幅逐渐变小。
- 适用条件：固定频率高于阈频，光路和材料条件稳定；真实平台可能受仪器量程、空间电荷和电极结构影响。
- 与项目仿真的关联：饱和平台要影响数值曲线、动画收集率和讲解面板，不应被实现成无限线性增长。
- 后续审查要检查的实现点：`experiment.js` 是否有饱和电流上限或平台模型；`mapping.js`/`voltAmpTrajectory.js` 是否把高正压区映射到平台而非持续线性增大。
- 来源文件路径：`knowledge/claims.md` C002、C003；`knowledge/topics/T02-light-intensity-saturation-current.md`；`knowledge/topics/T03-volt-ampere-curve.md`；`knowledge/papers/P009.md`；`knowledge/papers/P025.md`。
- 证据等级：A。
- 是否需要论文原文复核：需要平台数值参数时复核。

### K009 反向电压、截止电流与弱信号拖尾

- 理论名称：反向减速与截止附近拖尾。
- 简要说明：反向电压抑制光电子到达阳极；接近截止电压时电流通常应连续趋近低值，实际读数可能受初速度方向分布、本底/暗电流和仪器阈值影响。
- 对应规律：反向电压越强，收集率越低；理想截止点与最大初动能对应，但测量曲线可能需要外推或阈值判定。
- 适用条件：动画可用平滑函数和概率收集表达宏观趋势；不要求逐电子真实量子轨迹。
- 与项目仿真的关联：微观动画可展示电子回退、少量越过势垒、统计窗口和低电流噪声，但不得把截止点处理成无条件瞬时跳变。
- 后续审查要检查的实现点：`physics.js`、`voltAmpTrajectory.js`、`engine.js` 是否处理反向场、回退和低电流状态；`dataDiagnostic.js` 是否允许合理外推。
- 来源文件路径：`knowledge/claims.md` C003、C004、C006；`knowledge/topics/T03-volt-ampere-curve.md`；`knowledge/topics/T04-electron-motion-and-collection.md`；`knowledge/papers/P024.md`；`knowledge/papers/P032.md`。
- 证据等级：A。
- 是否需要论文原文复核：需要具体拖尾形状时复核。

### K010 微观动画的概率化收集和初速度分布

- 理论名称：宏观电流的统计化动画表达。
- 简要说明：单个电子轨迹与宏观电流不是一一对应关系，教学动画可以用概率化收集、初速度扰动、轨迹淡出和统计窗口表达宏观趋势。
- 对应规律：发射事件密度和收集率共同决定统计电流表现；电子初速度大小/方向分布会影响能否被收集。
- 适用条件：该条是教学仿真表达，不是完整量子过程或真实电子束模拟。
- 与项目仿真的关联：动画引擎不必逐电子严格求解真实三维分布，但核心趋势必须与频率、光强、电压和材料关系一致。
- 后续审查要检查的实现点：`engine.js` 粒子生成、电子步进、收集统计；`physics.js` 初速度和电场力方向；`mapping.js` 的 `collectionRatio` 和发射率。
- 来源文件路径：`knowledge/claims.md` C004；`knowledge/topics/T04-electron-motion-and-collection.md`；`knowledge/papers/P021.md`；`knowledge/papers/P024.md`；`knowledge/papers/P032.md`；`README.md` 的微观动画说明。
- 证据等级：A。
- 是否需要论文原文复核：不需要作为教学简化依据；需要真实分布参数时复核。

### K011 本底电流、暗电流和阳极寄生项

- 理论名称：弱电流测量中的寄生电流/本底项。
- 简要说明：本底电流、暗电流、阳极光电流或其他寄生项会干扰截止电压和弱光电流判断，适合在动画和报告中作为误差来源说明。

- 适用条件：项目级误差模型，未必对应精确仪器标定值。

### K012 光阑、距离、调零和仪器误差
