# T05 仪器因素、误差与数据处理

## 主题用途

用于约束光阑、距离、仪器状态、数据拟合和误差来源在实验流程中的表达。

## 可直接采用的项目结论

- 光阑大小、光电管接收距离和仪器条件会改变有效入射光通量与实验误差，需要在实验台和报告中作为独立影响因素呈现。（支撑：P022、P023、P026、P027）
- 普朗克常量测定不应只给出单点读数，应支持外推法、曲率法、线性回归或软件辅助处理等多种数据处理解释。（支撑：P001、P003、P004、P005、P006、P013、P015、P020）

## 对项目模块的约束

- VirtualLab.vue
- ExperimentReport.vue
- useVirtualLabController.js
- 报告生成
